Installation On A Laptop

R and RStudio are separate piece of software that both need to be installed. RStudio will check for a working R installation, so please install R first.

R can be downloaded and installed from CRAN. Please choose the version for your operating system (OS)

RStudio can be downloaded and installed Posit with options for most operating systems.

Once you have R installed, please open either R (or the Console pane in RStudio if you’re confident) and paste the following then hit the Enter key to begin the package installation.

install.packages(c("tidyverse", "rmarkdown"))

You will need an internet connection and the installation may take up to 10 minutes. A large amount of text will scroll past you as the installation takes place. However, if it gives you a message informing you that something has failed, please keep reading or attend one of the drop-in sessions for help.

Using A Remote Desktop

Posit Cloud

If you do not have access to any of the above a cloud instance is available from Posit as a last resort. Once you have registered, go to Your Workspace and select New Project from the menu at the top RHS of the page, choosing “New R Studio Project”.

In the Console section of the screen, please run the installation code from above. This should not fail as Posit has developed both the tidyverse and rmarkdown and the code has been tested.
